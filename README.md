Perfume Store Website

A modern and responsive perfume store website developed using HTML and CSS. This project showcases a simple e-commerce landing page design with an intuitive layout and user-friendly navigation.

Features
Responsive navigation bar
Search functionality UI
Product showcase section
About Us section
Contact information section
Clean and elegant design
Technologies Used
HTML5
CSS3
Purpose

This project was created to practice front-end web development concepts, including page layout design, responsive styling, and website structure.
